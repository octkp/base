# 専門性

**目標**

- BigAdvanceの課金アーキテクチャに関する深い理解に基づき、年払い機能を主導する

**取り組み**

- 月払い・年払い・契約更新に関する設計から実装、リリースまでを主導
- 課金パターンの検証設計を行い、品質を担保

**成果**

- 年払い機能のリリース完了
