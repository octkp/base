# 目標達成

## 1. 年払い機能のリリース

**目標**

- 年払い機能をスケジュール通りにリリースする

**取り組み**

- フロント、バックの2名体制で実装を進めた
- PdMやQAとも密に連携を取りつつ、仕様変更にも柔軟に対応しつつ進めた

**成果**

- 2025/10/23 ~ 2026/1/13（約3ヶ月）ほどの長期間にもかかわらず、遅延なくスケジュール通りにリリースを完了
- リリース後も障害なく、安定して稼働中
