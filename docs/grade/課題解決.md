# 課題解決

## 1. 年払い機能の品質担保

**課題**

- 仕様が複雑（月払い⇔年払い変更、契約更新等のパターンが多数）かつ、テストコードの整備が困難な状況

**取り組み**

- AIエージェントを活用し、課金パターンのテストデータ生成スキルを独自開発 → テスト準備工数を大幅に削減し、全パターンの網羅検証を実現
- 十分な準備期間を確保するため、開発着手前に検証計画を策定

**成果**

- スケジュール通りリリース、リリース後の重大不具合ゼロ

## 2. 環境構築スクリプトの作成

**課題**

- 環境構築が複雑、かつスムーズに完了しないことが多い

**取り組み**

- 環境構築を最初から最後まで実施してくれるスクリプトを作成し、展開した

**成果**

- 環境構築がスクリプト一発で実施でき、環境構築による工数削減に貢献した

## 3. 属人化による学習機会の損失リスク

**課題**

- 属人化による学習機会の損失リスク

**取り組み**

- ペアプロとソロ開発+AIエージェント活用を比較検討
- 現チームのスキルレベルとAIエージェントの活用度を踏まえ、ソロ開発+AIレビュー+ナレッジ共有のプロセスが最適と判断し提案
- Claude Codeを使用し、ナレッジを簡単に記録できる仕組みを構築

**成果**

- Gr内にナレッジ蓄積の文化を定着させ、⚪︎件の調査・実装ナレッジをドキュメント化
- 過去の調査経緯を参照可能にしたことで、類似課題の再調査工数を削減
