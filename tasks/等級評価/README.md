## 概要

等級評価（G4・シニアメンバー）の自己評価文を4軸（目標達成・課題解決・組織貢献・専門性）で作成・レビュー・改善した。

### ゴール

4つの軸について、それぞれ3つの項目を挙げ、各項目が定性的・定量的に測れる状態であること。

## 結論/解決策

5回のレビューサイクルを経て、全4軸A-以上に到達。

### 最終構成

**目標達成（A）**: 年払いリリース / 定期リリース安定運用 / メンバー目標達成支援

**課題解決（A）**: 年払い品質担保 / 環境構築スクリプト / 属人化リスク解消

**組織貢献（A-）**: リポジトリオーナー / CI/CD整備 / postgres障害対応

**専門性（A-）**: 課金ドメイン / 商談依頼 / 金融機関統合

### 重要な改善ポイント
- 課題解決: 定量データの充実（半日→30分、35件のナレッジ等）
- 組織貢献: CI/CD差し替えで3種のフィードバック形態をカバー
- 専門性: 全項目に「メンバー育成への貢献」を追加しG4要件を網羅
